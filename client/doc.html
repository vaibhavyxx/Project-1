Documentation of Library API

API

All Books

  • Relative URL: /getData
  • Supported Methods: GET, HEAD
  • Query Params: None
  • Body Params:
  • Endpoint returns and its format: JSON object
    Format: {author: string, country: string, language: string, link: string, pages: number, title: string, year: number, genres: string}
  • Example:
    • Request: /getData
    • Result: [{ "author": "Chinua Achebe", "country": "Nigeria", "language": "English", "link": "https://en.wikipedia.org/wiki/Things_Fall_Apart\n", "pages": 209, "title": "Things Fall Apart", "year": 1958, "genres": [ "Historical Fiction", "Postcolonial Literature" }]

Get Country

  • Relative URL: /getCountry
  • Supported Methods: GET, HEAD
  • Query Params: country - the place of origin of the book
  • Body Params:
  • Endpoint returns and its format: JSON object
    Format: {author: string, country: string, language: string, link: string, pages: number, title: string, year: number, genres: string}
  • Example:
    • Request: /getCountry?country=India
    • Result: [ { "author": "Kālidāsa", "country": "India", "language": "Sanskrit", "link": "https://en.wikipedia.org/wiki/Abhij%C3%B1%C4%81na%C5%9B%C4%81kuntalam\n", "pages": 147, "title": "The recognition of Shakuntala", "year": 150, "genres": [ "Historical Fiction", "Classical Literature" ] }]

Add Books

  • Relative URL: /addBook
  • Supported Methods: POST
  • Query Params:
  • Supported Body Formats: application/json, application/x-www-form-urlencoded
  • Endpoint returns and its format: JSON object
    Format: {author: string, link: string, title: string, year: number, genres: string}
  • Example:
    • Request: /addBook
    • Body: {author: 'author1', link: 'link1', title: 'title1', year: 'year', genres: 'genres1, genres2'}
    • Result: {author: 'author1', link: 'link1', title: 'title1', year: 'year', genres: 'genres1, genres2'}

Add More Genres

  • Relative URL: /addGenres
  • Supported Methods: POST
  • Query Params:
  • Supported Body Formats: application/json, application/x-www-form-urlencoded
  • Endpoint returns and its format: JSON object
    Format: {title: string, genres: string}
  • Example:
    • Request: /addGenres
    • Body: {title: 'title', genres: 'genre3'}
    • Result: {author: 'author1', link: 'link1', country: 'country', language: 'language', pages: 'pages', title: 'title', year: 'year', genres:['genres1', 'genres2', 'genre3']}
